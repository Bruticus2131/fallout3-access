# fallout3-access
an accessibility mod for fallout 3
Fallout3-access is an accessibility mod that allows blind players play fallout3.
It supports keyboard nawigation and it has screen reader support via tolk
Features: Now the mod has menu reading and it supports the following game menus: main menu, settings menu, pip boy, but experimental, pausemenu, save and load menu. 
To do: add more menus accessibility, world nawigation.
How to install: 1. buy fallout 3 on steam You can find it there https://store.steampowered.com/app/22300/Fallout_3/
2. When you buy the game download the latest fose version under this link: https://fose.silverlock.org
Download the fose v1.3 beta 2. 
What is fose? Fose is an script extender, that expands scripting capabilities of the game, you can create c++ plugins etc.  
3. download the anniversary pather from nexus mods, because you need to downgrade the game, because fose doesn't support the latest version, just type in google fallout anniversary pather nexus mods. 
You need to create  an account there to download it. 
4. When you download fose and fallout anniversary pather, unpack the .zip files and paste all the contents of annivarsarypather.zip and fosev1.3 beta2.zip to the root fallout3 folder. Run the pather.exe and wait for it to finish, when it will finish pathing press enter. 5. Finally, download the fallout3-access.zip from the releases page of this repo. And paste all contents of it to the fallout3\data
